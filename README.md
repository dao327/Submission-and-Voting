# User Work Submission and Voting Page

This is a static front-end website prototype.

## Main Features

- Submit creative works
- Upload work images
- Display work cards
- Vote for works
- Live ranking list
- Search function
- Category filter
- Sort by newest or most votes
- Save data using localStorage

## How to Use

1. Unzip the downloaded file.
2. Open `index.html` in a browser.
3. Test the submission and voting functions directly.

## Important Note

This is a front-end prototype. The data is saved only in the current browser.

To turn it into a real multi-user website, you would need:

- A back-end system
- A database
- User login or device verification
- Anti-vote-spam protection
- Admin review functions
